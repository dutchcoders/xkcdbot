XKCDbot

Introduction

The XKCD bot will surprise you with random XKCD images.

Configure your Slack Integrations

  • Create a new Slash Command
    • Name the command /xkcd
    • Set the URL to be https://xkcdbot.herokuapp.com/.
    • Method should be POST
    • Add some usefull information and usage information
  • Create a new Incoming Webhook
    • Name the command /keybase
    • Set the URL to be https://xkcdbot.herokuapp.com/.
    • Method should be POST
    • Add some usefull information and usage information
Usage

Ok, now you can test the integration. Type 

/xkcd
in Slack, and you'll get the response.
